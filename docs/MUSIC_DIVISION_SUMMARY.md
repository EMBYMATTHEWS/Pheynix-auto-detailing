### Priority 1: Distribution (Week 1)
1. **Choose distributor:** DistroKid (recommended)
2. **Upload first single:** 4-6 weeks before release
3. **Claim Spotify for Artists:** https://artists.spotify.com/
4. **Create social media profiles:** @emby.matthews everywhere

### Priority 2: Content (Weeks 1-2)
1. **TikTok:** Post daily teasers (15-30 sec clips)
2. **Instagram:** Behind-the-scenes content
3. **YouTube:** Upload lyric video first
4. **Email list:** Start building from day 1

### Priority 3: Marketing (Weeks 3-4)
1. **Playlist pitching:** Submit to 20+ playlists
2. **Social media campaign:** Use hashtags #newmusic #emby_matthews
3. **Fan outreach:** Direct message 100 fans/artist profiles
4. **Ads budget:** Start with $100 for Instagram/TikTok
